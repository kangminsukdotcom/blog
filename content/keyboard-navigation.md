Yes, this website supports keyboard shortcuts.

이 웹사이트는 키보드 단축기를 지원합니다.

```
- H: Home
- B: Blog
- E: English
- K: Korean
- N or ⇧+N: Navigation
- ← or →: Prev or next post

- H: 홈
- B: 블로그
- E: 영어
- K: 한국어
- N or ⇧+N: 페이지 내 이동
- ← or →: 이전, 다음 글
```
